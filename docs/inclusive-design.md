# Inkluzivni dizajn

Projekt implementira najmanje dvije komponente inkluzivnog dizajna:

1. **High-contrast mode**
   - prebacuje vizualnu temu na visoki kontrast (crna pozadina / bijeli tekst)
   - pomaže korisnicima s oštećenjem vida i u situacijama slabog osvjetljenja

2. **Povećanje fonta (font scale slider)**
   - globalno povećava font preko CSS varijable `--font-scale`
   - pomaže korisnicima s potrebom za većim fontom bez zoomanja cijele stranice

Dodatno:
- `skip-link` omogućuje korisnicima tipkovnice brzi skok na glavni sadržaj.
- `:focus-visible` stil osigurava vidljiv fokus pri navigaciji tipkovnicom.
